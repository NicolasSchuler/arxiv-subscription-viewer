Features

Everything you need to stay on top of arXiv papers without leaving your terminal

Fuzzy Search

Rapidly find papers by title or author with fuzzy matching powered by RapidFuzz

Smart Organization

Tags, stars, notes, read/unread tracking, and watch lists for authors and keywords

LLM Summaries

Generate AI-powered summaries and relevance scores using Claude, Codex, or any LLM CLI

Semantic Scholar

Citation counts, TLDRs, similar paper recommendations, and citation graph exploration

Export & Download

BibTeX, Markdown, RIS, CSV export, batch PDF downloads, and clipboard support

Beautiful Themes

Monokai, Catppuccin, Solarized Dark, and High Contrast themes with category highlighting and no-color/ASCII fallbacks

History Mode

Auto-discover and navigate between arXiv email archives by date

HuggingFace Trending

Community upvotes, AI summaries, keywords, and GitHub info from HF Daily Papers

Vim-Style Navigation

j/k navigation, marks, search bookmarks, and keyboard-first workflow

Quick Start

Get up and running in under a minute

1

Install

$ pip install arxiv-subscription-viewer
2

Add (or auto-save) arXiv emails

$ mkdir -p history
# Save digest as history/2026-02-13.txt (manual or via mail rule)

Tip: automate this step by writing each digest to history/YYYY-MM-DD.txt.

No-email mode: arxiv-viewer --api-category cs.AI

3

Run

$ arxiv-viewer

Accessibility: arxiv-viewer --color never --ascii

Non-interactive mode: arxiv-viewer --list-dates
